# claude

This repo holds overarching context that applies across all of the sibling repos under `/home/gsej/repos/`. Repo-specific `CLAUDE.md` files in each sibling repo override or extend anything here.

## Purpose

- Shared context that applies to every repo (preferences, conventions, ways of working).
- Reusable rules, samples, and scripts that may be useful in other repos.

## Layout

As content is added, organise it by kind rather than by repo:

- `rules/` — guidance and conventions that apply across repos.
- `samples/` — reference snippets and examples to copy from.


Per-repo specifics live in that repo's own `CLAUDE.md`, not here.
